sql 查询小技巧

2019-05-26
	mysql 在WHERE 对比时间查询的时候可以用空字符串。但是做比较时，只能大于空字符穿不能小于空字符串，
	因为当小于空字符串查询的时候是不会查询到内容的，
	大于空字符穿儿的话就相当于没有加上这个 WHERE 条件。

SELECT
	 *
FROM
	table
WHERE
	id = '123123'
AND updateTime > ''

	使用场景：mybatis-plus 用注解sql 不写SQL mapper 文件（sql mapper 写起来嫌麻烦）
	java 中date类型进行查询的时候，防止 date为null ，可以在代码里判断date是否为null。
	如果为null的话，可以返回一个空字符串给sql，这样就能满足正常的需求
