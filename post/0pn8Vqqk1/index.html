数据结构--链表

2019-05-26

linked：链表->单链表

本次示例实现的是单向链表

  1. 简述：链表是一种常用的组织有序数据的数据结构，它通过指针将一系列数据节点连接成一条数据链， 是线性表的一种重要实现方式。相对于数组，链表具有更好的动态性，建立链表时无需预先知道数据总量， 可以随机分配空间，可以高效地在链表中的任意位置实时插入或删除数据。链表的开销主要是访问的顺序 性和组织链的空间损失。

  2. 单链表图解：

  • 单链表是最简单的一类链表，它的特点是仅有一个指针域指向后继节点（next），因此，对单链表的遍历只能从头至尾（通常是NULL空指针）顺序进行。
  • 每一个数据块儿都包含了两个数据：当前域存储的数据、下一个域的指针
  1. 其核心方法与list类似：
   get(index); 根据索引位置获取元素
   add(value);往链表内添加元素
   remove(index); 根据索引位置删除元素
   size(); 获取链表总长度
  1. 展示java代码的具体实现

public class ILink {


    /**总长度*/
    private int size;
    /**链*/
    private Node node;

    public void add(Object value){
        Node node = new Node(value);
        if(this.node == null){
            this.node = node;
        } else {
            this.node.add(node);
        }
        size ++;
    }

    public Object get(int index){
        if(index > size -1){
            return null;
        }
        int i =0;
        if(i == index){
            return this.node.value;
        } else {
            return this.node.next.get(index, i);
        }
    }

    public boolean remove(int index){
        if(index > size -1){
            return false;
        }
        int i = 0;
        if(i == index){
            this.node = this.node.next;
        } else {
            this.node.remove(index, i);
        }
        return true;
    }

    public int getSize(){
        return this.size;
    }

    public void innerObj(){
        System.out.println("输出链表的内容======= start");
        Node node = this.node;
        while (node.hashNext()){
            System.out.print(node.getValue() + "，");
            node = node.next;
        }
        System.out.println(node.getValue());
        System.out.println("输出链表的内容=======  end");
    }

    private class Node{

        private Object value;
        private Node next;

        public Node(){}

        public Node(Object value){
            this.value = value;
        }

        public boolean hashNext(){
            return this.next != null;
        }

        public void add(Node node){
            if( hashNext()){
                this.next.add(node);
            } else {
                this.next = node;
            }
        }

        public Object get(int index, int currentIndex){
            currentIndex ++;
            if(index == currentIndex){
                return this.value;
            }
            return next.get(index, currentIndex);
        }

        public boolean remove(int index, int currentIndex){
            currentIndex ++;
            if(index == currentIndex){
                this.next = this.next.next;
            } else {
                this.next.remove(index, currentIndex);
            }
            return true;
        }

        private Object getValue(){
            return this.value;
        }
        public void setValue(Object value){
            this.value = value;
        }
    }
}
